# Soul Layer Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 实现初始灵魂画像生成、持久化读取，以及 `openbiliclaw profile` 的真实展示闭环。

**Architecture:** 新增 `ProfileBuilder` 负责画像 prompt、LLM 结果校验和 `SoulProfile` 构建；`SoulEngine` 负责读取偏好层、调用 builder、保存与读取 `soul.json`；CLI 只负责展示或引导。整个实现保持 TDD，先锁定 JSON 解析、存储行为和 CLI 输出，再补最小实现。

**Tech Stack:** Python 3.11, Typer, Rich, dataclasses, pytest, Ruff, mypy
